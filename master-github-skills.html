        ChromeDevTools/chrome-devtools-mcp
        
Official Google MCP Server · 31.6k ★ · 1.9k forks

        
Gives Claude Code full control of your real Chrome browser — with your existing cookies, logins, and sessions intact. Click, type, fill forms, navigate, screenshot, inspect network traffic, run Lighthouse audits. No headless detection, no login walls, no bot blocking. This solves the #1 problem across T1 (Shopify automation), T3 (Eventbrite/UtahValley scraping), and any project that needs to interact with a live website.

        

          T1 Shopify Admin
          T3 Event Scraping
          Authenticated Sites
          Browser Automation
          Screenshot Output
        

      

      

        
Install once — works everywhere:

        

          

            1
            Paste into Claude Code (Code tab):
          

          Write {"mcpServers": {"chrome-devtools": {"command": "npx", "args": ["-y", "chrome-devtools-mcp@latest"]}}} to C:\Users\Ben\AppData\Roaming\Claude\claude_desktop_config.json
          

            2
            Restart Claude Desktop
          

          

            3
            Test: ask Claude Code to navigate to any URL and describe what it sees
          

          

            
            Mark as installed
