T4 Ludwig-Integrated Eight-Week Proposal

Proposal only. Active syllabus, database schema, arcs 2-8 and class generator are not replaced by this page.

Practical Direction

Keep the hands-first eight-week cycle. Use Ludwig as a progression and command-round influence, Easton as the mechanics backbone, FightClub for New and Exciting, Sylvie only for selective beginner-appropriate depth, and Bookends for warm-ups and burnouts.

Proposed Revised Structure

Week 1 -- Beginner Foundation: stance, guard, jab-cross, teep exposure, rear-kick exposure

Required core

  • stance
  • guard
  • jab
  • cross
  • jab-cross on targets
  • basic teep target contact
  • rear-kick target contact

Interchangeable options

  • single-punch call rounds
  • two-punch call rounds
  • short walking kick line reps
  • simple reset-after-strike challenge

Three-night arc

  • Night 1 stance/guard plus jab-cross on mitts
  • Night 2 jab-cross review plus lead/rear teep on shields
  • Night 3 hand/teep review plus rear round kick and scaled check-return

Prior knowledge: None. This is the front door.

Rolling-entry adaptation: Teach the base version of every weapon on a target. First-day students do not need prior terminology.

Recurring review: None from prior weeks; build stance and guard into every round.

Progression destination: Coach-called jab-cross, teep or kick target round using four progressive 3-minute New and Exciting rounds.

FightClub options: Power-transfer jab-cross, teep reaction shield drill, rear-kick shield application

Ludwig concepts: number-call drilling, single/two-punch assembly, cumulative round progression

Practical change: Keep the current technique-first Week 1, but make progression explicitly cumulative instead of three isolated intro nights.

Week 2 -- Jab, Cross and Jab-Cross Precision

Required core

  • head-level jab/cross mechanics
  • pad holding
  • guard return
  • straight-line target accuracy

Interchangeable options

  • single-punch series
  • two-punch series
  • double-jab entry
  • basic number calls 1/2 using WCS mapping

Three-night arc

  • Night 1 single-punch mechanics
  • Night 2 two-punch series and double jab
  • Night 3 coach-called jab-cross rounds

Prior knowledge: Week 1 stance/guard and first target-contact experience.

Rolling-entry adaptation: Re-teach stance and jab/cross mechanics before any command round.

Recurring review: Week 1 stance, guard, reset, target discipline.

Progression destination: Jab-cross pad round with simple number-call variation.

FightClub options: mitt relay, power-transfer challenge, reaction pad touch

Ludwig concepts: single-punch series, two-punch series, number-call drilling

Practical change: Use Ludwig-style call rounds to increase variety without changing Arc 2 mechanics.

Week 3 -- Punching With Movement and Basic Defense

Required core

  • step-in/step-out punching
  • parry/block basics
  • return after defense

Interchangeable options

  • slip introduction
  • roll exposure only if hook mechanics are stable
  • short defend-and-return number calls

Three-night arc

  • Night 1 step with punches
  • Night 2 parry/block and return
  • Night 3 movement plus defense-return

Prior knowledge: Jab-cross and target reset.

Rolling-entry adaptation: Run all defense as announced feeds at touch speed.

Recurring review: Jab-cross, stance, guard, reset.

Progression destination: Announced feed: punch, defend, return jab-cross, exit.

FightClub options: reaction mitt drill, beginner defense-return application

Ludwig concepts: gradual defense introduction, command rounds that retain prior punch segments

Practical change: Keep defense narrow but use call structure so review is active instead of lecture.

Week 4 -- Teep and Range Control

Required core

  • teep mechanics
  • shield target contact
  • hand-to-teep transition
  • range reset

Interchangeable options

  • walking teep line
  • front-kick number call
  • basic teep defense exposure

Three-night arc

  • Night 1 lead teep
  • Night 2 rear teep and walking reps
  • Night 3 jab-cross-teep command round

Prior knowledge: Stance, guard, jab-cross.

Rolling-entry adaptation: Begin with shield target rules and lead teep only.

Recurring review: Jab-cross and distance reset.

Progression destination: Coach-called jab-cross-teep range round.

FightClub options: teep reaction shield drill, distance challenge

Ludwig concepts: walking kick progression, number-call hand-to-kick transition

Practical change: Use teep as the first clean punch-to-kick bridge before round-kick week.

Week 5 -- Rear Round Kick Foundation

Required core

  • rear round kick mechanics
  • pad/shield contact
  • balance and reset
  • basic check exposure from frozen Arc 1

Interchangeable options

  • walking round kicks
  • jab-cross-low kick descriptive combo
  • kick-count command rounds

Three-night arc

  • Night 1 rear kick mechanics
  • Night 2 walking/stepping kick progression
  • Night 3 basic check-return from frozen Arc 1

Prior knowledge: Stance, jab-cross, teep.

Rolling-entry adaptation: Teach rear kick from scratch; use prior tools only as optional entries.

Recurring review: Jab-cross and teep range.

Progression destination: Jab-cross to rear kick or check-return on shield.

FightClub options: rear-kick shield drill, kick accuracy challenge

Ludwig concepts: walking round kick progression, punch-to-kick assembly

Practical change: Do not disturb Arc 1; add Ludwig influence only as optional sequencing around it.

Week 6 -- Checks, Low Kicks and Returns

Required core

  • round-kick check
  • low-kick target contact
  • defense-and-return
  • controlled partner feed

Interchangeable options

  • standard Dutch-style cooperative exchange
  • block-return command rounds
  • simple punch-or-kick read

Three-night arc

  • Night 1 low kick and check
  • Night 2 return after defense
  • Night 3 constrained punch/kick read

Prior knowledge: Rear kick and basic punch defense.

Rolling-entry adaptation: Use announced feeds and static role assignment.

Recurring review: Week 3 punch defense and Week 5 rear kick.

Progression destination: Announced attack, defend, return on target.

FightClub options: kick/check exchange, defense-return reaction drill

Ludwig concepts: standard Dutch drilling as cooperative exchange, block/check command rounds

Practical change: Restore Dutch drilling as usable standard format, while keeping power and roles beginner-safe.

Week 7 -- Combination Building and Basic Knees

Required core

  • combination assembly
  • basic straight knee
  • marching knee
  • pad holder roles

Interchangeable options

  • walking knees
  • coach-called 2-3 weapon rounds
  • descriptive punch-body-kick combinations

Three-night arc

  • Night 1 punch-to-kick assembly
  • Night 2 straight knees and walking knees
  • Night 3 multi-tool command round

Prior knowledge: Hands, teeps, kicks, checks.

Rolling-entry adaptation: Teach the knee as a standalone target tool first; shorten combinations for first-day students.

Recurring review: All prior weapons, especially jab-cross and rear kick.

Progression destination: Jab-cross-kick-knee on pads or a shorter beginner version.

FightClub options: pad combination challenge, knee target drill

Ludwig concepts: cumulative combination assembly, walking knee progression, number-call rounds

Practical change: Use Ludwig-style assembly to make combinations feel connected instead of a list of combos.

Week 8 -- Integration and Controlled Application

Required core

  • review of all tools
  • standard Dutch-style exchange option
  • defense-return integration
  • capped technical round for ready students

Interchangeable options

  • number-call review rounds
  • Dutch drill with limited weapons
  • fixed-role substitute for new students

Three-night arc

  • Night 1 command review
  • Night 2 cooperative exchange and returns
  • Night 3 capped constrained round or substitute drill

Prior knowledge: Weeks 1-7.

Rolling-entry adaptation: First-day students use fixed-role review drills, not the technical round.

Recurring review: Full cycle.

Progression destination: Controlled attack-defend-return sequence and one capped coach-refereed technical round.

FightClub options: reaction/combo challenge, high-energy review game

Ludwig concepts: Dutch-style cooperative partner drilling, command review, cumulative sequence retention

Practical change: Make Week 8 more systemized with command/Dutch formats while preserving the existing safety cap.
