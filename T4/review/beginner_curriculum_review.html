Summary

curriculum_id: wcs-beginner-striking-foundation-v1

Competencies

Required

  • Athletic Muay Thai stance
  • Basic guard
  • Forward / backward / lateral movement + pivot
  • Jab, Cross, Jab-Cross
  • Lead teep, Rear teep
  • Rear round kick
  • Step-in rear round kick
  • Basic low kick Easton evidence is generalized within Round Kicks; low-target application leans on FightClub.
  • Basic knee (straight / step)
  • Basic punch-to-kick combinations
  • Basic check (leg check)
  • Basic punch defense (parry / block)
  • Simple defense-and-return pattern
  • Pad-holding fundamentals Demonstrate/Detail/Drill (D.D.D.) is Easton's own pad-holding coaching-method video.
  • Distance management
  • Balance / recovery to stance
  • Controlled partner reaction drills FightClub-led; no Easton Kickboxing equivalent, Easton Fundamentals transcription may add depth.

Optional exposure

  • Lead hook, basic uppercut exposure Folded into the general 'Punches' video, not individually taught in depth -- exposure only.
  • Basic lead-leg kick / switch-kick foundation Explicitly deferred from Arc 1 (too advanced for absolute first day); reintroduced as optional exposure once stance/balance are solid.
  • Basic teep defense No Easton Kickboxing video; FightClub-only support.
  • Feints A nuance layered onto already-mastered strikes, not core to first mastery.

Deferred material

  • Advanced lead-leg kick variations (switch/step/spot/jump, full set)
  • Jump switches, complex stance switching
  • Catch-and-trip systems
  • Sweeps
  • Elbows as a major beginner unit (Easton has 4 elbow videos -- reserved for post-foundation)
  • Dirty boxing, advanced clinch (Easton Kickboxing has zero clinch content)
  • Open sparring
  • Complex slip and bob-and-weave chains (conflicts with the check-based defense system this foundation teaches)
  • Advanced kick catches
  • Complex long combinations
  • Advanced counter systems

Week-by-Week Map

Week 1: Base Rear Round Kick built and approved

Required: Athletic Muay Thai stance, Basic guard, Rear round kick, Step-in rear round kick, Basic check (leg check), Balance / recovery to stance

Terminal week goal: Step in with a rear round kick, recover in balance, check a partner's controlled counter kick, and return immediately with a round kick.

  • Night 1: stance, weight shift, step angle -> base rear round kick mechanics -> terminal pad drill.
  • Night 2: compressed review (base kick, now on partner pads) -> step-in entry mechanics -> terminal step-in pad drill.
  • Night 3: compressed review (step-in kick) -> check mechanics -> check-and-return -> final assembled terminal drill.

Night 4 (reinforcement) / Night 5 (review-application): same skills, different drills.

Week 2: Jab-Cross Foundation proposed

Required: Jab, Cross, Jab-Cross, Distance management

Terminal week goal: Throw a clean jab-cross combination while stepping, on pads, both partners rotating.

  • Night 1: stance/guard recap -> jab mechanics -> cross mechanics, shadow reps then pad intro.
  • Night 2: jab-cross combination -> step-with-punch footwork (L-Step / Pivot Step).
  • Night 3: compressed review (jab-cross) -> terminal partner pad drill: jab-cross with footwork.

Night 4 (reinforcement) / Night 5 (review-application): same skills, different drills.

Week 3: Teep Foundation proposed

Required: Lead teep, Rear teep

Terminal week goal: Chain a jab-cross into a teep, on pads, with clean distance control.

  • Night 1: lead teep mechanics on pads.
  • Night 2: rear teep + alternating-teep footwork.
  • Night 3: compressed review (jab-cross) -> terminal: jab-cross-teep on pads.

Night 4 (reinforcement) / Night 5 (review-application): same skills, different drills.

Week 4: Basic Combinations: Punch-to-Kick proposed

Required: Basic punch-to-kick combinations

Terminal week goal: Throw jab-cross-rear kick as one connected combination, on pads, at full arc-terminal intensity.

  • Night 1: 2-punch-to-kick combos (jab-cross-rear kick).
  • Night 2: teep-to-punch and punch-to-teep transitions.
  • Night 3: compressed review (teep) -> terminal: signature 3-strike combo (jab-cross-rear kick) live on pads.

Night 4 (reinforcement) / Night 5 (review-application): same skills, different drills.

Week 5: Midpoint Integration & Application proposed

Required: (none new -- integration/graduation)

Terminal week goal: Apply the full Weeks 1-4 toolkit live, under light time pressure, at capped power.

  • Night 1: mixed pad rounds cycling kick/teep/punch combos + partner reaction games.
  • Night 2: constrained 2-choice decision drills built only from already-taught responses.
  • Night 3: terminal -- an assembled, coach-refereed, capped-power technical round using only already-taught tools.

Night 4 (reinforcement) / Night 5 (review-application): same skills, different drills.

Open question: Night 3's 'technical round' format depends on Ben's answer to whether controlled technical sparring belongs in the beginner cycle at all -- see Open Questions. Fallback if no: replace Night 3 with a second constrained decision-drill round.

Week 6: Basic Defense Systems: Punch & Kick proposed

Required: Basic punch defense (parry / block)

Terminal week goal: Correctly read and defend either a punch or a kick attack, then return fire, on a live (but constrained) partner feed.

  • Night 1: punch defense -- parry/block fundamentals.
  • Night 2: kick defense -- check reinforcement + low-kick checks.
  • Night 3: compressed review -> terminal: defense-and-return decision drill (partner throws punch OR kick, student reads and responds).

Night 4 (reinforcement) / Night 5 (review-application): same skills, different drills.

Week 7: Basic Knees & Close-Range Introduction proposed

Required: Basic knee (straight / step)

Terminal week goal: Add a straight knee onto an existing punch combination, live on pads.

  • Night 1: straight/marching knee mechanics, standing and on pads.
  • Night 2: knee-to-combination integration (adding the knee onto already-taught punch combos).
  • Night 3: compressed review -> terminal: knee application pad drill.

Night 4 (reinforcement) / Night 5 (review-application): same skills, different drills.

Week 8: Foundation Graduation: Assembled Attack-Defend-Counter proposed

Required: (none new -- integration/graduation)

Terminal week goal: Run a complete attack -> defend -> counter exchange using the full beginner toolkit, live, at capped power.

  • Night 1: broad but shallow review sweep across all major tools (an intentional exception to 'not a survey', justified by the graduation context).
  • Night 2: assembled attack sequence -- student initiates with a punch/kick combo.
  • Night 3: TERMINAL -- full assembled attack-defend-counter live drill: Partner A attacks with a combo, Partner B defends (check or parry) and returns.

Night 4 (reinforcement) / Night 5 (review-application): same skills, different drills.

Notation Proposal

No existing numbered strike notation was found in the WCS competency material (it lists techniques by name, not number), in the Easton Kickboxing transcripts, or in current project records/coach terminology. The an outside reference gym's materials DO use numbered punch/combo codes internally, which is a structural reason to be cautious about Option C in particular, independent of the numbering's technical merits.

Option A -- Technique IDs only

Canonical technique IDs and structured combo steps; no compact numeric notation required.

Pros
  • Zero ambiguity -- every step traces directly to a database technique_id
  • No numbering-conflict risk with any external system (WCS or otherwise)
  • Simplest possible generator parsing
Cons
  • Not coach-facing shorthand -- a coach can't call out 'jab-cross-kick' as '1-2-RK' on the floor without a separate cue sheet
  • Verbose for combo display

Option B -- Numeric notation as coach-facing aliases

Canonical technique IDs remain primary in storage. Numbers are a display-only alias layer and compact combo notation for coach-facing output.

Pros
  • Coach adoption is easy -- numbers are how most gyms already call combos out loud
  • Technique IDs stay the source of truth, so orthodox/southpaw and generator logic never depend on numbering
  • Numbering scheme can be swapped or gym-customized without touching stored data
Cons
  • Two representations to keep in sync (ID and alias)
  • Requires an explicit mapping table (which numbers mean what) to avoid coach confusion

Option C -- Numeric notation as schema spine

Numbered notation becomes the primary combo representation; everything else derives from it.

Pros
  • Maximally compact combo storage
  • Matches the numbering convention many boxing/Muay Thai coaches already think in
Cons
  • Punch numbering (1=jab, 2=cross, etc.) does not extend cleanly to kicks/knees/elbows without an ad-hoc extension scheme
  • Orthodox vs. southpaw meaning of a number can silently flip if not handled explicitly
  • Directly risks colliding with an outside reference gym's own numbering conventions -- given that outside gym's content must never appear in output, building WCS's core schema around a numbering style that resembles theirs is a real confusion/citation risk, not just a technical one
  • Hardest to migrate away from later if it turns out inadequate, since the whole schema depends on it
Decision owner: Ben -- not decided automatically by this architecture pass.
